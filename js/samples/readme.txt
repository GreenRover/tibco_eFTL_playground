The Publisher JavaScript automatically publishes messages to the 
sample destination on the specified channel every 5 seconds. 

The Subscriber JavaScript application subscribes to the sample
destination on the specified channel and prints the received messages.

In order to run the JavaScript samples copy the sample *.html files
and eftl.js into the same directory and open the *.html file using
a browser.
